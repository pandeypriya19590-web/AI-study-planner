# AI-study-planner
This project uses Python and Machine Learning (Linear Regression) to predict how many study hours a student needs for upcoming exams. The program takes inputs like mid-sem marks, days left for exam, and target marks, and then provides a basic study recommendation.

Technologies Used Python Pandas NumPy Scikit-Learn

How to Run python study_planner.py

Purpose A simple ML project for beginners to understand data preprocessing, model training, prediction, and result visualization
